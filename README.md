
# SuperSonic Ready App

Applicazione iOS educativa pronta per essere evoluta in release App Store.

## Stack
- SwiftUI
- Vision OCR
- AVFoundation
- MVVM Architecture

## Features
- OCR da fotocamera
- Sillabazione italiana
- Evidenziazione sillabe
- Voce italiana
- UI bambino-friendly
