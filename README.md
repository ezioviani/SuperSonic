
# SuperSonic Web

Web app educativa installabile su iPhone tramite Safari.

## Deploy GitHub Pages

1. Carica i file su GitHub
2. Vai su Settings → Pages
3. Deploy from branch
4. Seleziona main
5. Salva

URL finale:
https://ezioviani.github.io/SuperSonic/

## Installazione su iPhone

Apri con Safari:
Condividi → Aggiungi a Home
